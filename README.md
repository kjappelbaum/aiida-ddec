# aiida-ddec 

a ddec plugin for AiiDA

# Installation

```shell
pip install git+ https://github.com/yakutovicha/aiida-ddec
verdi quicksetup  # better to set up a new profile
verdi calculation plugins  # should now show your calclulation plugins
```
